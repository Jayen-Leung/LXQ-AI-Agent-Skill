# Manuscript Revision Output Contract

Use `strict` intensity.

Return:

1. Stable editor/reviewer comment IDs.
2. Faithful concern summary and severity.
3. Evidence or author input required.
4. Proposed action and actual manuscript location.
5. Draft response that does not claim unperformed work.
6. Cross-artifact consistency and remaining limitations.

Scientific and numerical corrections precede language polish. Recheck every downstream table, figure, supplement, and response touched by a change.
