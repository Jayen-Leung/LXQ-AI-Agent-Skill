# Literature Report Output Contract

Use `standard` for a focused report and `strict` for a formal evidence audit.

Return the review question, search scope and date, exact query or its location, selection boundary, study evidence matrix, source anchors, effect and uncertainty, bias/limitation, contradictory evidence, support grade, full-text status, and lawful acquisition provenance.

Do not call an informal search systematic, treat metadata as full-text evidence, or imply unavailable papers were read.
