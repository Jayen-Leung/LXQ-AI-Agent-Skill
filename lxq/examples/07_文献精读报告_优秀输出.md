# 文献精读报告：优秀示例

## 任务输入

精读一篇预测模型论文，判断能否支持临床应用主张。

## 调用强度

`strict`

## 任务模式

`audit`、`explain`

## 功能轨道

`literature`、`analysis`

## 推荐输出结构

论文身份 → 问题与设计 → 数据/模型 → 主要结果 → 图表证据 → 偏倚与适用性 → 支持等级。

## 最终示范输出

论文报告的是单中心回顾性模型开发，训练与验证均来自同一时间段。模型区分度较好，但未报告外部校准，特征缺失处理是否在数据拆分后完成也不清楚。因此，该论文可以支持“模型在内部数据中显示预测信号”，不能支持“模型已经具备临床推广价值”。

关键核验点包括样本筛选流程、事件数、特征数、调参是否嵌套、校准图、决策曲线和真实外部队列。每项结论应附页码、段落、图号或表号来源锚点。

## 质量检查点

- 区分作者主张和证据能支持的结论。
- 不以期刊级别或引用次数替代质量评价。
- 保留图表和正文来源锚点。
